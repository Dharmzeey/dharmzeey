Selenium

Using Selenium I developed a web scraping bot that fetches user data of students of my school I was able to accomplish this by taking the matric numbers of students and writing it twice.This was possible because our portal is wack, most students use their matric number as username and password. Also when we input our details twice and it is incorrect, it will reset to matric number as username and password after 24h hours so I exploited that opportunity and ran a script that feeds the input with matric number twice, if successful, it will go to the personal details and fetch all the details. But if not, it will log the matric number that was reseted onto and excel file, which I can then rerun the script after 24 hours. I also took care of other errors like old password and student whose profile are not up-to-date
